В папке VacationHost находится приложение которое следует запустить от имени администратора. 
Данное приложение хостит сервис. Также в этой папке находится база данных.
В папке VacationClient находится клиент.
В базе есть три аккаунта:
admin admin - HR
user user - сотрудник
guest guest - сотрудник

Что должно было быть сделано, если бы это был рабочий продукт:
 - хранить пароли в базе в виде хэша sha1(sha1(пароль))
 - ssl-соединение клиента и сервера
 - проверка того что дата окончания позже даты начала отпуска (в этом варианте количество дней выводится отрицательным числом)
 - юнит-тесты
 - комментарии
 - система контроля версий (git/svn)
 - виртуализация datagridview при большом количестве строк
 - доработки дизайна (запрет изменения размеров некоторых форм, ...)
 - профилирование нагрузки
 - логирование и отправление отчета об ошибках (если таковые будут) с разрешения пользователя
